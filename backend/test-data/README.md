# Test Data

Test files and results from extraction testing.

## Files

- **`extraction_results.json`** - Results from running the ExtractionTester on sample receipts
- **`test-invoice-pattern.txt`** - Sample text for testing invoice pattern matching
- **`test_invoice_pattern.csx`** - C# script for testing invoice patterns

## Usage

These files are used by the test scripts in `scripts/` and the testing projects (`ExtractionTester`, `OcrTest`).

For sample receipt files (PDFs and images), see `../../docs/samples/`.
